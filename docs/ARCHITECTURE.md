# Архитектура приложения CardKeeper (черновик)

## Цель
Чистая, масштабируемая архитектура для учебного проекта, соответствующая современным стандартам Android-разработки.

## Выбранный стек (MVP)
*   **Язык:** Kotlin
*   **Архитектура:** MVVM (Model-View-ViewModel)
*   **Локальная БД:** Room
*   **Асинхронность:** Kotlin Coroutines + Flow
*   **DI (Dependency Injection):** Ручная инжекция
*   **Работа с камерой:** CameraX
*   **Генерация штрихкодов:** Библиотека `com.google.zxing:core`
